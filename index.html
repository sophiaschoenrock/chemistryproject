About

This project is a collection of static HTML/CSS/JS pages that load a shared questions.json bank. Students can practice by unit and subsection, see why each option is right or wrong, and use a fast-paced review mode when they want a challenge.

It is built for anyone preparing for AP Chemistry coursework or exams and for teachers who want a simple, hostable practice site without a backend.

Features

Everything runs in the browser; deploy to GitHub Pages and share one URL.

  • Units 1–9 filtering

    Pick a unit or drill a subsection; the quiz filters questions by CED-style topic prefixes.

  • Rationales for A–D

    After you answer, read explanations for every choice — not just the correct letter.

  • Reaction Rush mode

    Timed rounds with score and streak; optional unit focus before you dive in.

  • Static & portable

    No build step for the web pages — drop the folder on any static host.

  • Curriculum landings

    Unit overview pages link into filtered quizzes (e.g. Unit 1 topic grid).

  • Import pipeline

    Python script merges external MCQ JSON into the shared bank with deduplication.

Getting started

Clone the repository, serve the folder over HTTP (browsers block fetch() to JSON from file://), and open the pages locally. 

git clone https://github.com/sophiaschoenrock/chemistryproject.git
cd chemistryproject
python3 -m http.server 8080
# Visit http://localhost:8080/

On GitHub Pages, the site root serves index.html automatically once Pages is enabled (see instructions below).
